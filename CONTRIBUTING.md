# How to Contribute to NeuralBlitz

First off, thank you for considering contributing. NeuralBlitz is a project built on the principle of symbiotic co-evolution, and your involvement is crucial to its growth and success.

### Code of Conduct

All contributors are expected to read and adhere to our [Code of Conduct](CODE_OF_CONDUCT.md). Please ensure you are familiar with its contents. We are committed to fostering an open, welcoming, and respectful environment.

### Ways to Contribute

#### Reporting Bugs

*   If you find a bug, please check the [issues](https://github.com/NuralNexus/NeuralBlitz/issues) to see if it has already been reported.
*   If not, please [open a new issue](https://github.com/NuralNexus/NeuralBlitz/issues/new?template=bug_report.md) using the "Bug Report" template. Provide as much detail as possible, including steps to reproduce the issue.

#### Suggesting Enhancements

*   Have an idea for a new feature, a new Capability Kernel (CK), or an improvement to the architecture? We'd love to hear it.
*   Please [open a new issue](https://github.com/NuralNexus/NeuralBlitz/issues/new?template=feature_request.md) using the "Feature Request" template to start a discussion.

#### Pull Requests

1.  **Fork the repository** and create your branch from `main`.
2.  **Install dependencies** (`pip install -r requirements.txt`).
3.  **Make your changes.** Please adhere to the style guides and ensure all new code is accompanied by tests.
4.  **Run the tests** to ensure everything is passing.
5.  **Submit a pull request.** Provide a clear description of the problem and your solution. Link to any relevant issues.

### Styleguides

*   **Code:** We follow the [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide for Python.
*   **Documentation:** All documentation should be clear, concise, and written to be accessible. Avoid unnecessary jargon where possible.
*   **Git Commits:** Please follow a conventional commit message format (e.g., `feat: Add new ReflexaelCore tuner`).

Thank you for helping us build the future of symbiotic intelligence.
